# Neuroimaging Methods Lab

A public portfolio of hands-on neuroimaging projects built to develop practical skills in fMRI analysis, computational neuroscience, and reproducible scientific workflows.

## Mission

This organisation documents a structured training journey through core neuroimaging methods. Each repository represents applied learning through implementation, analysis, and interpretation rather than passive coursework alone.

## Core Areas

* Resting-state functional connectivity
* Default mode network analysis
* Task-fMRI and GLM-based activation modeling
* Predictive modeling from neuroimaging data
* Machine learning classification
* Scientific computing with Python
* Reproducible research practices

## Why This Organization Exists

The goal of this lab is to turn theory into practice: building technical fluency, research discipline, and a strong foundation for future work in computational neuroscience and brain research.

Five completed portfolio repositories, with advanced expansion projects in development.
